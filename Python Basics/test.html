2) Paragraph

هذا مثال على فقرة نصية باستخدام عنصر <p>. يمكنك وضع أي نص هنا لعرضه كمحتوى عادي داخل الصفحة.

6) Inline Elements

العناصر Inline لا تبدأ سطرًا جديدًا بشكل افتراضي:

<span> <strong> <em> <a>

Block Elements

العناصر Block تحتل كامل العرض وتبدأ سطرًا جديدًا:

<div> كتلة 1
<p> كتلة 2
<section> كتلة 3
